# مجله فشن ماسا

مرجع آموزشی فشن، طراحی لباس و سبک زندگی — تهیه‌شده توسط [آموزشگاه ماسا](https://masa.ac) شیراز، زیر نظر دانشگاه شیراز.

## اجرا

```bash
pnpm install
pnpm dev
```

سایت روی [localhost:3000](http://localhost:3000) بالا می‌آید و به `/docs` هدایت می‌شود.

## بیلد

```bash
pnpm build
pnpm start
```

خروجی استاتیک در پوشه `out` قرار می‌گیرد.

## محتوا

مقالات در `content/docs/` هستند — هفت فصل از مبانی فشن تا فشن پایدار.
