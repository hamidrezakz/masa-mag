# پارچه‌ها و متریال‌ها

انتخاب پارچه یکی از تعیین‌کننده‌ترین تصمیم‌های طراح است. دو لباس با طرح کاملاً یکسان — اما با پارچه‌های متفاوت — می‌توانند از زمین تا آسمان فرق داشته باشند.

<Callout title="پارچه: روح لباس">
  وقتی می‌گوییم «این لباس دست می‌افتد» یا «خیلی خشک است» داریم از پارچه صحبت
  می‌کنیم. شناخت پارچه برای هر طراحی پایه‌ای‌ترین مهارت است.
</Callout>

---

## پارچه‌های طبیعی

<Tabs items={["پنبه", "ابریشم", "پشم", "کتان"]}>
  <Tab value="پنبه">
    **منشأ:** الیاف گیاه پنبه **ویژگی‌ها:** - نرم و راحت برای پوشیدن - تنفس‌پذیر
    — مناسب آب‌وهوای گرم - قابل شستشو با ماشین - کمی چروک می‌شود - قیمت مناسب
    **انواع پارچه‌های پنبه‌ای:** - **Jersey:** نرم، کشسان — مناسب تی‌شرت،
    لباس‌های راحتی - **Denim:** سخت، بادوام — جین، جکت - **Poplin:** ظریف،
    براق‌کمی — پیراهن رسمی - **Muslin:** سبک، نیمه‌شفاف — نمونه اولیه لباس -
    **Canvas:** ضخیم، محکم — کفش، کیف **مناسب برای:** پوشاک روزانه، تابستانی،
    بچگانه، Basics
  </Tab>
  <Tab value="ابریشم">
    **منشأ:** پیله کرم ابریشم **ویژگی‌ها:** - براق و لوکس - سبک و لطیف -
    تنفس‌پذیر اما در عین حال گرم‌کننده - به مراقبت دقیق نیاز دارد - قیمت بالا
    **انواع پارچه‌های ابریشمی:** - **Charmeuse:** یک طرف براق، یک طرف مات — برای
    لباس‌های شب، لانژری - **Chiffon:** شفاف، سبک — لایه‌بندی، اسکارف -
    **Dupioni:** بافت ناهموار — برای لباس عروس، مجلسی - **Crepe de Chine:** مات،
    سنگین‌تر — بلوز، دامن **مناسب برای:** لباس‌های شب، مجلسی، لانژری، اسکارف،
    لباس عروس **ایران:** ابریشم ایرانی، به‌ویژه از کاشان و مازندران، شهرت تاریخی
    جهانی دارد.
  </Tab>
  <Tab value="پشم">
    **منشأ:** الیاف پشم گوسفند، شتر، بز کشمیری و... **ویژگی‌ها:** - گرم‌کننده
    عالی - تنفس‌پذیر - اگر مراقبت درست داشته باشد، بادوام است - ممکن است خارش
    ایجاد کند (به جز Merino و Cashmere) **انواع:** - **Merino Wool:** پشم ظریف
    با کمترین خارش - **Cashmere:** از بز کشمیری، لطیف‌ترین و گران‌ترین -
    **Tweed:** ضخیم، با بافت نامنظم — کت، کاپشن - **Flannel:** نرم و تراشیده —
    شلوار رسمی، پیراهن پاییزه **مناسب برای:** پاییز/زمستان، کت، کاپشن، کفش، شال
  </Tab>
  <Tab value="کتان">
    **منشأ:** الیاف گیاه کتان **ویژگی‌ها:** - تنفس‌پذیرترین پارچه طبیعی - در
    گرما بسیار خنک - با گذشت زمان نرم‌تر می‌شود - چروک می‌شود — اما «چروک کتانی»
    خودش یک استیتمنت است! - قیمت متوسط تا بالا **مناسب برای:** تابستان، پوشاک
    ساحلی و مدیترانه‌ای، پیراهن‌های مردانه، کت‌های سبک **ایران:** کتان در
    آب‌وهوای گرم جنوب ایران بسیار مناسب است.
  </Tab>
</Tabs>

## پارچه‌های مصنوعی

<Accordions>
  <Accordion title="پلی‌استر (Polyester)">
    **منشأ:** مواد پتروشیمی **ویژگی‌ها:** - بسیار بادوام - کمتر چروک می‌شود -
    قیمت پایین - تنفس‌پذیری کم — در گرما ناراحت - محیط‌زیست: مشکل‌ساز
    (میکروپلاستیک) **کاربرد:** Fast Fashion، ورزشی، آستر لباس **توجه:** بیش از
    ۵۰٪ پارچه‌های دنیا امروز پلی‌استر است.
  </Accordion>
  <Accordion title="نایلون (Nylon)">
    **ویژگی‌ها:** - بسیار مقاوم و کشسان - خشک می‌شود سریع **کاربرد:** جوراب،
    لباس زیر، کوله‌پشتی، لباس‌های ورزشی
  </Accordion>
  <Accordion title="اسپندکس / لایکرا (Spandex/Lycra)">
    **ویژگی‌ها:** - فوق‌العاده کشسان — تا ۶ برابر طول اصلی کشیده می‌شود -
    برمی‌گردد به شکل اصلی **کاربرد:** معمولاً به پارچه‌های دیگر اضافه می‌شود
    (۲-۵٪) برای کشش. جوراب، لباس ورزشی، شنا.
  </Accordion>
  <Accordion title="ویسکوز / ریون (Viscose/Rayon)">
    **منشأ:** نیمه‌طبیعی — از سلولز چوب درخت ساخته می‌شود. **ویژگی‌ها:** - احساس
    طبیعی مانند ابریشم - ارزان‌تر از ابریشم - تنفس‌پذیر - با شستشوی زیاد ممکن
    است آسیب ببیند **کاربرد:** لباس‌های روزانه، بلوز، دامن — جایگزین ارزان‌قیمت
    ابریشم
  </Accordion>
</Accordions>

## نحوه خواندن برچسب پارچه

هر لباسی برچسبی دارد که ترکیب پارچه را نشان می‌دهد. درک این برچسب‌ها مهم است:

| علامت                     | معنی                                |
| ------------------------- | ----------------------------------- |
| 100% Cotton               | کاملاً پنبه                         |
| 60% Cotton, 40% Polyester | ترکیب — معمولاً راحت‌تر و کمتر چروک |
| W 🌊                      | قابل شستشو با دست                   |
| 🌀 30°                    | ماشین با حداکثر ۳۰ درجه             |
| 🚫🌀                      | فقط خشکشویی                         |
| ☀️                        | می‌توان در آفتاب خشک کرد            |

<Callout title="انتخاب پارچه برای طراحی">
  قبل از انتخاب پارچه از خودتان بپرسید: این لباس کجا پوشیده می‌شود؟ هوا چقدر گرم
  یا سرد است؟ پوشنده چقدر فعال است؟ بودجه چقدر است؟ پارچه درست پاسخ این
  سوال‌هاست.
</Callout>
